# Project Intent: IA para Projetos Culturais

## What
A web application to manage, analyze, and improve cultural project proposals using artificial intelligence. The platform allows users to create, review, and optimize projects for cultural funding (e.g., Lei Rouanet), providing tools for project scoring, budget management, and compliance with public funding requirements.

## Why
To democratize access to cultural funding by empowering project creators with intelligent tools that increase the quality, competitiveness, and impact of their proposals. The platform aims to simplify the process, provide actionable feedback, and help users maximize their chances of approval.

## Current State

- Project detail view with scoring, budget, and checklist
- Mock data for projects and sections
- UI components for forms, cards, progress, and more

## Current Features

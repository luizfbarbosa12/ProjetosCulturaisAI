# Feature: Project Listing and Navigation

## What
Allows users to view a list of their cultural projects, navigate to project details, and create new projects.

## Why
To provide users with an organized overview of all their projects and quick access to manage or review each one, supporting efficient project management and tracking.

## Acceptance Criteria
- [ ] Users can see a list of all projects
- [ ] Users can navigate to a project detail page
- [ ] Users can create a new project

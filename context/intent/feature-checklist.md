# Feature: Project Checklist

## What
Provides a checklist for each project to track completion of required items for funding applications.

## Why
To help users ensure all necessary steps and documentation are completed, reducing the risk of missing requirements.

## Acceptance Criteria
- [ ] Users can view and check off required items
- [ ] Checklist progress is visible in project detail
- [ ] Checklist is linked to project sections

# Feature: Project Detail and Scoring

## What
Displays detailed information for a selected project, including sections, scores, budget, and checklist. Allows users to review and analyze each aspect of their project.

## Why
To help users understand the strengths and weaknesses of their projects, improve proposal quality, and ensure all requirements are met for funding applications.

## Acceptance Criteria
- [ ] Users can view all sections of a project
- [ ] Users can see project scores and suggestions
- [ ] Users can review and manage budget items
- [ ] Users can track checklist completion
